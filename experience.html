Introduction/Problem Statement

Over the years, Wisconsin Information Technology (IT) students find it difficult to excel in their exams unlike in other departments. With the fast-paced advancements made within the technology sector, they struggle to meet the skill set requirements. Learning courses such as C++, Java, Probability, etc., can be very tedious at times, because they require more coding practice and more related learning materials. As a consequence, students often resort to relying on lengthy internet searches for similar learning resources. The availability of learning materials can have salient effects on students’ performance. This is where WizzyGeeks comes in handy. WizzyGeeks is more than just another technology learning website because it is customized just for Wisconsin International University College (WIUC) Students. The basis of this site is for students to hone Computer Science and Information Technology computing skills. The Website will keep track of the School’s curriculum in this related field; this means it will have solutions to exercises and past questions on related courses. In addition, WizzyGeeks aims at training students to meet current industrial needs with sufficient content on programming with Java and C++.

Requirements of WizzyGeeks

The expectation of this website ranges from providing solutions to class exercise problems to advance learning of the two core programming languages of the school (C++ and Java). Below are what the site should offer:

  1. It should be able to have a registration platform that will also allow users to login after registration, and to help store their information on the database.
  2. It should be user friendly; every piece of information is displayed openly.
  3. It should contain a full course of C++ programming and the class exercises with their respective solutions attached of various levels.
  4. It should also contain a full course Java programming guide and the class exercise with their solutions attached of various levels.
  5. It should contain all exercises and solutions of Algebra, Discrete Maths, Probability and Statistics, and Numerical Methods.
  6. It should have an article that details how the internet works.

Objectives

User interfaces for software applications can come in a variety of formats, ranging from command-line, graphical, web application, and even voice. While the most popular user interfaces include graphical and web-based applications, occasionally the need arises for an alternative interface. Whether due to multi-threaded complexity, concurrent connectivity, or details surrounding execution of the service, a chat bot based interface may suit the need. Chat bots typically provide a text-based user interface, allowing the user to type commands and receive text as well as text to speech response. Chat bots are usually a stateful services, remembering previous commands (and perhaps even conversation) in order to provide functionality. When chat bot technology is integrated with popular web services it can be utilized securely by an even larger audience.

SCOPE

  • A Student Information Chat Bot project is built using artificial algorithms that analyzes user’s queries and understand user’s message.
  • This System is a web application which provides answer to the query of the student very effectively. .
  • Students just have to query through the bot which is used for chatting.
  • Students can chat using any format, as there is no specific format that the user has to follow.
  • The System uses built in artificial intelligence to answer the query.
  • The answers are appropriate what the user queries.
  • If the answer found to be invalid, user just need to select the invalid answer button which will notify the admin about the incorrect answer.
  • Admin can view invalid answer through portal via login
  • System allows admin to delete the invalid answer or to add a specific answer of that equivalent question.
  • The user does not have to personally go to the college for enquiry.
  • This system helps the student to be updated about the college activities.
